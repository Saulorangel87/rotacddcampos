# 📍 Mapeamento Digital CDD Campos dos Goytacazes

![Status do Projeto](https://img.shields.io/badge/Status-Em%20Desenvolvimento-yellow)
![GitHub License](https://img.shields.io/github/license/saulorangel87/saulorangel87.github.io)

O **Mapeamento Digital** é uma ferramenta web desenvolvida para modernizar a logística de distribuição postal. O objetivo central é **substituir os antigos mapas de papel** por uma interface interativa, dinâmica e acessível via dispositivos móveis para os carteiros em rota.

---

## 🎯 Objetivo do Projeto

Tradicionalmente, a orientação geográfica nos distritos de entrega dependia de mapas físicos que se desgastam e dificultam a atualização. Este site integra a API do Google Maps com camadas personalizadas para:
- Otimizar o deslocamento dos profissionais.
- Facilitar a identificação de limites entre distritos.
- Centralizar ferramentas úteis (Busca CEP e Rastreamento) em um único local.

## 🚀 Funcionalidades

- **Menu Interativo (Hambúrguer):** Navegação otimizada para dispositivos móveis (Mobile First).
- **Camadas de Distritos:** Separação visual de 9 setores (Distritos 601 a 609).
- **Integração com Google My Maps:** Visualização precisa das rotas de entrega em Campos dos Goytacazes.
- **Acesso Rápido:** Atalhos para o site oficial dos Correios e ferramentas de logística.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica.
* **CSS3:** Layout moderno utilizando **Flexbox** e **Grid**, com **Media Queries** para total responsividade.
* **JavaScript (Vanilla):** Lógica de interação para o menu e controle de camadas.
* **Font Awesome:** Iconografia profissional.
* **GitHub Pages:** Hospedagem e Deploy contínuo.

## 📱 Interface Mobile

O projeto foi pensado para o uso "em campo". Por isso, a interface se adapta automaticamente a smartphones, garantindo que o menu de distritos não atrapalhe a visualização do mapa durante o trabalho.
